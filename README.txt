RIME西米 for Squirrel 说明

版本 1.0（维护中）

功能概述：
1. 自动配色：输入主色即时生成完整主题（深/浅模式、WCAG 对比度保证可读性）
2. 布局与方向：stacked（垂直）/ linear（水平自适应宽度），horizontal / vertical
3. 预编辑模式：inline_preedit 动态调整组字区位置
4. 候选嵌入：inline_candidate 切换候选词/拼音显示
5. 视觉样式：圆角、边框（-2 表示无边框）、透明度、阴影、候选项缩放
6. 间距控制：line_spacing（stacked 模式）、spacing（编码区边距）
7. 字体设定：全局/标签/注释三套独立字体
8. 候选格式自定义：[label] / [candidate] / [comment] 占位符
9. 即时预览：所有设置变更即时反映
10. 载入 YAML 主题文件：支持读取并编辑现有配色方案
11. 即时生成设定码：直接复制粘贴到 squirrel.custom.yaml

使用方式：
☞ https://gjrobert.github.io/Rime-See-Me-squirrel/
完成设置后，复制右栏"生成设定码"到 squirrel.custom.yaml 即可。

支持浏览器：Chrome 内核（Chrome、Vivaldi、……）或 Firefox 68+
